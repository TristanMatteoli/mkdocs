Description du site

  • Voici le lien pour accéder à notre site: https://a-pedagoarles-lamp.aix.univ-amu.fr/A2018M3104G20/login

  • Il y a un interface de connection fonctionnelle qui ramène directement sur l'accueil du site. Les indentifiants sont : "eleve@gmail.com" "testtest"

  • Le header permet d'afficher des icones de notifiactions (pour les mails reçu par l'étudiant ou des alertes), le nom, prénom et photo de l'utilisateur connnecté

  • La sidebar permet d'afficher les différentes sections du site (Accueil, Elements d'évalutation qui n'est pas fonctionnel, et Documentation qui ouvre la documentation MKDOCS)

  • D'un footer qui affiche le copyright

  • Dans le tableau de bord permet une recherche de stage ou de promotion. L'étudiant poura séléctionner grace a un menu déroulant le stage qu'il souhaite voir plus en détail.

  • Comme sur amétice, notre framework nous permet à l'utilisateur d'agencer les contenus du tableau de bord à sa façon.

  • L'utilisateur a un bouton pour accéder à son profil. Ce bouton est juste visuel car non fonctionnel.
